<decisions_made>

- **encodeDefaults=true**: kotlinx.serialization omits fields at their default value. ResourceId.kind="youtube#video" was never sent to the YouTube API. Fix: add encodeDefaults=true to the Json { } config in YouTubeModule.kt.
- **kind="youtube#playlistItem"**: Added to AddPlaylistItemRequest per YouTube Data API resource contract.
- **Scraping client separation**: @Named("scraping") OkHttpClient with no AuthInterceptor for EveryNoise scraping; default client with AuthInterceptor for YouTube API calls.
- **AuthFailureException ordering**: catch AuthFailureException before IOException in GeneratePlaylistUseCase so auth errors are not silently converted to NetworkError.
- **Looper fix**: AuthRepositoryImpl.refreshAccessToken() uses withContext(Dispatchers.Main) for AppAuth performTokenRequest (AppAuth creates Android Handler requiring a Looper on calling thread).
</decisions_made>
